PANDUAN LENGKAP URAT ID V2.0
============================

1. INSTALASI DATABASE
   - Upload semua file ke hosting (public_html).
   - Buka browser: https://domainanda.com/install_db.php
   - Jika sukses, HAPUS file install_db.php agar tidak di-reset orang lain.

2. KONFIGURASI ADMIN & AI
   - Buka file `config.php`.
   - Password Admin default: Amnet123 (sesuai request).
   - Masukkan GEMINI_API_KEY di `config.php` agar fitur AI Blog Generator berfungsi.
     (Ambil Key di: https://makersuite.google.com/app/apikey)

3. FITUR ADMIN (Login di /admin_login.php)
   - Approve Deposit: Cek bukti transfer, klik Approve. Saldo masuk otomatis ke user.
   - Blog Generator: Masukkan topik (misal: "Cara Youtuber Pemula Sukses"). AI akan membuat:
     a. Judul Clickbait
     b. Artikel 3000-5000 karakter (SEO Optimized)
     c. Gambar otomatis (menggunakan Unsplash API based on keywords)
   - Edit/Hapus Post: Sekarang tersedia di dashboard admin.

4. FITUR DEPOSIT
   - User memiliki 2 opsi: Instant (QRIS Generated) dan Manual (Bank Transfer).
   - QR Code di generate otomatis berdasarkan Rekening di config.php menggunakan API QRServer.

5. PANDUAN CRON JOB (Anti Unsub)
   - Agar user didenda jika unsubscribe, pasang Cron Job di cPanel.
   - Command: /usr/local/bin/php /home/username/public_html/cron_protect.php
   - Set waktu: Per Jam (0 * * * *).

6. TROUBLESHOOTING
   - Jika AI Error: Pastikan hosting support `curl` dan API Key benar.
   - Jika Gambar tidak muncul: Pastikan hosting support remote image fetching (allow_url_fopen).

Terima kasih.
